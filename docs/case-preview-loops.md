# Case Preview Loops

Diese Briefings sind fuer kurze, lautlose Vorschau-Loops in der Projekte-Uebersicht gedacht.
Empfehlung: 5-7 Sekunden, nahtlos loopbar, 16:9, dunkler Premium-Look, keine eingeblendeten Texte.

## 1. Case: Digitale Dokumentenplattform fuer Immobilien, Aktienregister und Broker

Use case: productivity-visual
Asset type: Website case-study preview loop
Primary request: Create a seamless 6-second cinematic loop showing a modern digital document platform in use for real estate, shareholder registry, and broker workflows.
Scene/backdrop: Dark premium SaaS workspace, floating document cards, secure folders, structured dashboard panels, subtle glass UI, warm-neutral lighting.
Subject: Digitized documents moving from chaotic paper stacks into a clean organized cloud workflow.
Composition: Medium-wide dashboard view with layered depth, animated document tiles sliding into categorized columns, soft camera drift, elegant motion.
Motion: Paper documents scan in, transform into digital cards, cards sort automatically into folders, status indicators turn from pending to verified, small progress pulses.
Style: High-end fintech SaaS, precise, minimal, trustworthy, modern Swiss business aesthetic.
Color palette: Charcoal black, graphite, warm white, subtle amber accents matching Datalization brand tone.
Loop behavior: End frame must visually match start frame for an infinite smooth loop.
Avoid: Cartoon look, clutter, exaggerated neon, people, visible brand names, fake gibberish text blocks.

Kurzfassung:
Chaotische Dokumente werden automatisch digitalisiert, sortiert und in eine strukturierte sichere Plattform ueberfuehrt.

## 2. Case: Digitale Loesung zur Uebermittlung von Steuerunterlagen

Use case: productivity-visual
Asset type: Website case-study preview loop
Primary request: Create a seamless 6-second loop showing tax and fiduciary documents being uploaded through a mobile-first secure submission workflow.
Scene/backdrop: Clean desk-to-mobile transition, smartphone upload interface, tax forms, receipts, scanned PDFs, secure transfer indicators, minimal professional background.
Subject: A frictionless app workflow where scattered tax documents become a complete structured submission.
Composition: Close-to-medium framing with a smartphone as the hero element, surrounding paper documents fading into digital order, subtle parallax motion.
Motion: Receipts and tax sheets lift from the desk, scan into the phone, transform into organized checklist items, completion ring reaches 100 percent, secure send confirmation appears without text.
Style: Elegant, efficient, reassuring, accounting-tech meets premium client portal.
Color palette: Off-black, soft grey, paper white, restrained amber highlight, realistic materials.
Loop behavior: Final state transitions naturally back into the initial scattered arrangement for a seamless repeat.
Avoid: Busy hands, visible faces, comic UI, unreadable fake dense text, bright playful colors.

Kurzfassung:
Steuerunterlagen werden per Smartphone sicher aufgenommen, geordnet und vollstaendig digital uebermittelt.

## 3. Case: Data Mining & Lead Intelligence fuer Immobilienverkaeufer

Use case: productivity-visual
Asset type: Website case-study preview loop
Primary request: Create a seamless 6-second data-mining preview loop for an automated real-estate lead intelligence platform.
Scene/backdrop: Dark analytics command center with maps, lead signals, filters, KPI counters, data streams, premium business intelligence interface.
Subject: Automated collection and qualification of seller leads from multiple sources.
Composition: Wide dashboard screen with map activity on the left and lead scoring panels on the right, cinematic angle, subtle camera movement.
Motion: Data points pulse onto a map, source streams flow into a central engine, selected leads rise into a qualified list, KPI counters increase smoothly, one highlighted lead card locks in with a subtle glow.
Style: Sophisticated BI platform, calm and intelligent, no hacker aesthetic, no chaos, enterprise-grade confidence.
Color palette: Deep black, slate, steel blue, muted white, restrained amber/orange accents for signals.
Loop behavior: The animation should resolve back into the initial dashboard state without a visible reset.
Avoid: Red alert visuals, cyberpunk, stock-market chaos, overly complex charts, tiny unreadable UI text.

Kurzfassung:
Mehrere Datenquellen fliessen in eine Lead-Engine, qualifizierte Immobilienverkaeufer werden automatisch identifiziert und priorisiert.
